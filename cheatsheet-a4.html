OpenWrt packages workshop cheatsheet

Patching

Prepare

Prepare pkg_name source code for patching: 
make package/pkg_name/{clean,prepare} QUILT=1 V=S
cd build_dir/target-arch_name/pkg_name

Create patch

Create new patch with quilt 
quilt new 010-main_code_fix.patch
quilt add  code.c	# add file
quilt edit code.c	# edit file
quilt diff		# check code change
quilt refresh		# save changes

Save patch

cd ~/openwrt_sdk_home
make package/pkg_name/update V=s
This will create patch in pkg_name/patches directory
